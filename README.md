About this repo
=========
I created for building the latest [libigl-python-bindings](https://github.com/libigl/libigl-python-bindings) every month.
The version will be `<year>-<month>` instead of semantic version of original source.

About igl
=========

Home: https://libigl.github.io

Package license: MIT

Feedstock license: [BSD-3-Clause](https://github.com/conda-forge/igl-feedstock/blob/master/LICENSE.txt)

Summary: Simple python geometry processing library

Development: https://github.com/libigl/libigl-python-bindings

Documentation: https://libigl.github.io/libigl-python-bindings/

libigl is a simple python and C++ geometry processing library.
We have a wide functionality including construction of
sparse discrete differential geometry operators and
finite-elements matrices such as the cotangent Laplacian and
diagonalized mass matrix, and
simple facet and edge-based topology data structures.
